# Especificación de Requisitos

## Requisitos Funcionales

RF01: Registrar usuarios.

RF02: Validar nombres y apellidos.

RF03: Registrar artículos.

RF04: Generar ID único para artículos.

RF05: Registrar préstamos.

RF06: Registrar devoluciones.

RF07: Generar certificados de devolución.

RF08: Generar facturas de venta.

RF09: Consultar estado de préstamos.

RF10: Exportar información a CSV.

RF11: Gestionar acceso de administrador.

RF12: Mostrar estadísticas generales.

## Requisitos No Funcionales

RNF01: El sistema debe desarrollarse en Python.

RNF02: Debe ejecutarse por consola.

RNF03: Debe almacenar información en archivos planos.

RNF04: Debe generar archivos TXT.

RNF05: Debe exportar información a CSV.

RNF06: Debe utilizar programación orientada a objetos.

RNF07: Debe utilizar las clases clsUsuarios y clsPrestamo.

RNF08: Debe ser fácil de usar.
