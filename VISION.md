# Reporte de Visión

## SIMPLEPRESTAMOS


## Descripción General

simplePrestamos es un sistema de gestión de préstamos desarrollado en Python para administrar usuarios, artículos, préstamos, devoluciones y ventas automáticas de objetos no devueltos.

## Problema

Michael Jackson Gamboa presta múltiples objetos a sus amigos, pero no recuerda quién tiene cada artículo ni cuándo debe ser devuelto.

## Solución

El sistema permitirá:

* Registrar usuarios.
* Registrar artículos.
* Registrar préstamos.
* Registrar devoluciones.
* Generar certificados.
* Generar facturas.
* Consultar reportes estadísticos.

## Beneficios

* Mejor control de inventario.
* Disminución de pérdidas.
* Automatización de procesos.
* Gestión eficiente de préstamos.
